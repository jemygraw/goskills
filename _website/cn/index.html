一应俱全

专为构建下一代 AI 代理的开发者设计。

技能解析器

轻松解析 `SKILL.md` 文件。根据官方规范提取元数据、指令和资源文件。

通用运行器

支持使用任意兼容 OpenAI 接口的大模型（如 GPT-4, DeepSeek 等）执行技能，而不仅仅是 Claude。

内置工具集

包含标准工具库：Shell 命令执行、Python 运行时、文件 I/O、网络搜索等。

MCP 集成

全面支持模型上下文协议（MCP），将您的代理连接到外部数据和服务。

开发者 CLI

使用 `goskills` 命令行工具直接在终端中检查、调试和运行技能。

纯 Go 实现

高性能、静态类型，易于部署。构建为可重用的 Go 模块，方便集成到您的项目中。

安装

几秒钟内即可运行。

Homebrew (macOS/Linux)

brew tap smallnest/goskills
brew install goskills

Go Install

go install github.com/smallnest/goskills/cmd/goskills@latest
go install github.com/smallnest/goskills/cmd/goskills-cli@latest

源码安装

git clone https://github.com/smallnest/goskills.git
cd goskills
make

使用指南

三个强大的工具，满足不同需求。

1. 运行器 (`goskills`)

模拟 Claude 技能使用工作流。它根据您的自然语言提示选择并执行技能。

# 1. 设置环境变量
export OPENAI_API_KEY="sk-..."
export OPENAI_MODEL="gpt-4o" # 可选，默认为 gpt-4o
export OPENAI_API_BASE="https://api.openai.com/v1" # 可选，默认为 OPENAI API URL


# 2. 执行任务（例如：生成艺术作品）
goskills run "创建一个生成抽象艺术的算法"

# 3. 使用自定义模型（例如：DeepSeek）
goskills run --model deepseek-v3 --api-base https://api.deepseek.com/v1 "您的提示词"

2. 下载器 (`goskills download`)

直接从 GitHub 仓库下载技能到您的本地技能目录。

# 从 GitHub 下载技能
goskills download https://github.com/ComposioHQ/awesome-claude-skills/tree/master/meeting-insights-analyzer

# 技能被下载到 ~/.goskills/skills
ls ~/.goskills/skills

3. 管理器 (`goskills-cli`)

检查、解析和验证您本地的技能定义。

# 列出目录中的所有技能
goskills-cli list ./skills

# 检查特定技能的结构
goskills-cli parse ./skills/my-new-skill

# 按关键词搜索技能
goskills-cli search ./skills "web scraping"
