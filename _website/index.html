Everything you need

Designed for developers building the next generation of AI agents.

Skill Parser

Effortlessly parse `SKILL.md` files. Extract metadata, instructions, and resource files according to official specifications.

Universal Runner

Execute skills with any OpenAI-compatible LLM (GPT-4, DeepSeek, etc.), not just Claude.

Built-in Tools

Includes a standard library of tools: Shell execution, Python runtime, File I/O, Web Search, and more.

MCP Integration

Full support for the Model Context Protocol (MCP) to connect your agents to external data and services.

Developer CLI

Inspect, debug, and run skills directly from your terminal with the `goskills` command-line tool.

Pure Go

High-performance, statically typed, and easy to deploy. Built as a reusable Go module for your projects.

Installation

Get up and running in seconds.

Homebrew (macOS/Linux)

brew tap smallnest/goskills
brew install goskills

Go Install

go install github.com/smallnest/goskills/cmd/goskills@latest
go install github.com/smallnest/goskills/cmd/goskills-cli@latest

From Source

git clone https://github.com/smallnest/goskills.git
cd goskills
make

Usage

Three powerful tools for different needs.

1. The Runner (`goskills`)

Simulates the Claude skill-use workflow. It selects and executes skills based on your natural language prompts.

# 1. Set up your environment
export OPENAI_API_KEY="sk-..."
export OPENAI_MODEL="gpt-4o" # Optional, defaults to gpt-4o
export OPENAI_API_BASE="https://api.openai.com/v1" # Optional, defaults to OPENAI api url



# 2. Run a task (e.g., generate art)
goskills run "create an algorithm that generates abstract art"

# 3. Use custom models (e.g., DeepSeek)
goskills run --model deepseek-v3 --api-base https://api.deepseek.com/v1 "your prompt"

2. The Downloader (`goskills download`)

Download skills directly from GitHub repositories to your local skills directory.

# Download a skill from GitHub
goskills download https://github.com/ComposioHQ/awesome-claude-skills/tree/master/meeting-insights-analyzer

# Skills are downloaded to ~/.goskills/skills
ls ~/.goskills/skills

3. The Manager (`goskills-cli`)

Inspect, parse, and validate your local skill definitions.

# List all skills in a directory
goskills-cli list ./skills

# Inspect a specific skill's structure
goskills-cli parse ./skills/my-new-skill

# Search for skills by keyword
goskills-cli search ./skills "web scraping"
